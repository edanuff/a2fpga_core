Wavetable Size Constants (from MAME):
------------------------------------
wavesizes[] = {256, 512, 1024, 2048, 4096, 8192, 16384, 32768}

Shift Formula:
-------------
curr_shift_w = 5'd9 + curr_res_w - curr_wts_w;

High Bit Formula (for overflow detection):
---------------------------------------
high_bit_w = 17 + resolution;

Accumulator Mask Formula:
------------------------
curr_acc_mask_w = {16'((1 << (9 + resolution)) - 1), 8'hFF};

Overflow Detection in MAME vs Verilog:
-----------------------------------
Verilog: overflow = temp_acc[high_bit_w];
MAME:    overflow = (altram >= wtsize);  // where altram = acc >> resshift
